on room start:
	- title slides in from right
	- main menu (play / credits / exit) slides in from left
	- menu active right away, does not wait for animation to finish
	
on exit:
	quit game immediately
	
on credits:
	- menu script sets room state to "room_change_requested"
	- menu script sets "target room" variable
	- controller checks room state
	- if room state is "room_change_requested", controller 
		- unpauses the title tween
		- sets room state to "breaking_down_room"
	- if room state is "breaking_down_room"
		- controller checks to see if title tween is complete
		- if title tween is complete, controller executes room_goto(target room)
